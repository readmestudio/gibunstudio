# 7일 내면 아이 찾기 프로그램

## 개요
7일간 데일리 미션을 수행하고 7일차에 분석 리포트를 받는 프로그램

## 가격
- 정가: 99,000원
- 무통장 입금 할인: 89,000원

## 프로그램 구조

### 데일리 미션
매일 감정일기 작성 (선택)

### 일자별 미션
- 1일차: TCI 검사
- 2일차: 어린시절 문장완성 (15문항)
- 3일차: 생각과 사고 구분하기 (33문항)
- 4일차: 핵심 신념 문장완성 (25문항)
- 5일차: Habit Mapper
- 6일차: 인지적 오류 검사 (15문항)
- 7일차: 리포트

## 일정
- 입금 확인일 = D1
- 12시 자정 기준 일자 업데이트
- 구매 후 14일 내 미션 완료 필요
- 1~6일차 미션 완료 시 리포트 생성 가능

## 리포트 조건
- 1~6일차 미션 모두 완료 필수
- 감정일기는 미완성이어도 가능
- 14일 내 미완료 시 리포트 미제공

## 추천 대상
- 같은 이유로 연인과 자주 다투는 분
- 반복되는 패턴의 연애를 하는 분
- 결혼을 앞두고 서로를 깊이 이해하고 싶은 커플
- 심리 상담을 받았지만 별 효용을 느끼지 못한 분

## 구현 상태
- 상세 페이지 구현 완료
- 일자별 미션 구현 완료
- 감정일기 LLM 연동 완료
- 리포트 생성 시스템 구현 완료
- Supabase 데이터 연동 대기
