# 1:1 심리 상담 프로그램

## 상담 종류 (1시간 기준)

### 1. 내면 아이 해석 상담
- 가격: 80,000원 → 70,000원
- 대상: 7일 내면 아이 찾기 프로그램 수료자만
- 내용: 7일 프로그램 결과 리포트 해석

### 2. 커플 내면 아이 상담
- 가격: 150,000원 → 140,000원 (2인)
- 대상: 둘 다 7일 프로그램 수료 필요
- 내용: 파트너와 함께 내면 아이 이해 및 관계 패턴 탐색

### 3. 패키지 상담
- 가격: 200,000원 → 190,000원
- 대상: 제한 없음
- 내용: 3가지 유료 검사 + 심층 분석 풀패키지

### 4. 1:1 개인 상담
- 가격: 100,000원 → 90,000원
- 대상: 제한 없음
- 내용: 이별, 애도, 불안, 우울 등 개인 고민

## 예약 플로우

### 1. 구매
상담 카테고리 선택 → 입금 → 코치 입금 확인 → 구매 활성화

### 2. 예약 요청
- 캘린더에서 코치 설정 가능 시간 확인
- 원하는 시간대 선택 (복수 가능)
- 서베이 제출 (고민, 목표)

### 3. 코치 확정
- 예약 요청 검토
- 시간 확정 + 줌 링크 설정
- 유저에게 알림

### 4. 상담 진행
- 확정된 시간에 줌 링크로 접속
- 상담일 2일 전까지 시간 변경 요청 가능
