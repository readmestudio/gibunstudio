# 결제 시스템

## 결제 방식
- **PG 미연동**: 무통장 입금만
- **은행**: 신한은행
- **계좌번호**: 140-015-244655
- **예금주**: 원모어 스푼

## 가격 정책

### 7일 내면 아이 찾기
- 정가: 99,000원
- 무통장 입금 할인: -10,000원
- 최종: 89,000원

### 1:1 상담
- 내면 아이 해석: 80,000원 → 70,000원
- 커플 내면 아이: 150,000원 → 140,000원
- 패키지 상담: 200,000원 → 190,000원
- 1:1 개인 상담: 100,000원 → 90,000원

## 결제 플로우

### 유저
1. 상세 페이지에서 "구매하기" 클릭
2. 결제 페이지: 정가 표시 → 할인 안내
3. 계좌번호 복사 기능
4. "입금 완료했어요" 클릭
5. 결제 완료 페이지 → 대시보드 이동
6. 입금 대기 상태

### 코치
1. 코치 모드 > 입금 대기 목록 확인
2. "입금 확인" 버튼 클릭
3. 입금 확인 시점 = D1 시작 (7일 프로그램)
4. 유저 구매 활성화
