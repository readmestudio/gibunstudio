# 7일 내면 아이 찾기 - 전체 구현 계획

## 기술 스택

- **프레임워크**: Next.js (App Router)
- **인증**: Supabase Auth (카카오 OAuth + 이메일)
- **DB**: Supabase (PostgreSQL)
- **배포**: Vercel
- **스타일**: Tailwind CSS
- **폰트**: Pretendard
- **LLM**: OpenAI GPT (감정일기, 리포트 생성)
- **알림**: 카카오 알림톡 (추후 구현)
