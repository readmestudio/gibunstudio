# 7일차: 리포트

## 리포트 생성 조건
- 1일~6일차 미션 모두 완료
- 감정일기는 미완성이어도 리포트 요청 가능
- 구매 후 14일 내 미션 미완료 시 리포트 미제공

## 리포트 양식

### 1. 검사 결과 프로파일
- 이름
- 검사 일시 (프로그램 시작일 ~ D+6)
- 결과 분석 보고자: 김연지 / 심리 상담사
- 유형: 동물 비유 + 친근한 이름 (예: 눈치 빠른 고양이)
- 유형 특징 요약 (5줄)

### 2. 지난 일주일간의 감정 분포도
- 감정일기 기반 감정 상태 시각화
- 일자별 막대 그래프 (감정별 색상 + 비율)

### 3. 이 아이는 이런 생각을 자주 해요
- 감정일기 자동 사고 분석
- 문장완성 검사 결과 반영
- 불릿 포인트 (400자 이내)

### 4. 이 아이는 나와 타인, 세상에 대해 이렇게 철썩같이 믿어요
- 어린시절 + 핵심 신념 검사 분석
- 나/타인/세상에 대한 믿음
- 불릿 + 세부 설명 (5600자 이내)

### 5. 이 아이는 왜 생겨났을까요?
- 문장완성, TCI, Habit Mapper 종합 분석
- 도표 (손그림 스타일)
- 상담사 설명 (줄글)

### 6. 이 아이는 나의 삶에 이렇게 영향을 미쳐요
- 건강한 상태 vs 스트레스 상태
- 관계(연인/가족/동료)에 미치는 영향

### 7. 상담사 총평
- 심리 상담사가 직접 작성

## 생성 방식
1. 코치가 로우데이터 확인
2. ChatGPT API로 1~6 섹션 자동 생성
3. 코치가 내용 편집 및 상담사 총평 작성
4. 퍼블리시 → 유저에게 전달 (알림톡/이메일)

## 구현 상태
- 리포트 템플릿 정의 완료
- ChatGPT 생성 API 구현 완료
- 코치 리포트 에디터 구현 완료
- 퍼블리시 플로우 Supabase 연동 대기
