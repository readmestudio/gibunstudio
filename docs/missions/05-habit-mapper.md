# 5일차: Habit Mapper

## 목적
나는 왜 안 좋은 습관을 반복할까?
습관의 구조(트리거 → 행동 → 결과)를 이해하고 탐색합니다.

## 활동 단계

### 1. 개념 설명
- 트리거: 습관을 시작하게 만드는 계기
- 행동: 실제로 반복하는 행동
- 결과(보상): 그 행동 뒤에 느끼는 감정이나 결과

### 2. 행동 작성
"내가 고치고 싶은 습관(행동)을 적어보세요"
예: 불안할 때 계속 휴대폰을 본다

### 3. 트리거 작성
"이 행동을 하게 만드는 트리거는 무엇인가요?"
(감정, 몸의 감각, 특정 상황)

### 4. 결과/보상 작성
- 단기적 보상
- 장기적 보상

### 5. 요약 및 성찰
"{트리거}일 때, {단기적 보상}을 위해 {습관 행동}을 반복하고 있군요."
"Habit Mapper를 완성하고 나니 어떤 생각이 드나요?"

## 마무리 메시지
"아주 중요한 한 걸음을 내디뎠어요. 습관을 반복하게 만드는 트리거를 아는 것만으로도 이 습관의 굴레를 끊어내는 과정은 이미 시작된 거예요."
