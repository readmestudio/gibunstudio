# MINDS · 5 ROLES — "내 안의 다섯 배역 찾기" 카드뉴스

`/minds` 무료 마음 진단으로 유입시키는 인스타 캐러셀 **7장** 시리즈.
다크 모노톤(`#0e0e10` + 오렌지 `#e85d3a`) — `instagram-cards/shared.css` 재사용.
흐름: **문제 인식 → 원리 → 다섯 배역 → 오해 풀기 → 불균형 사례 2개 → 진단 CTA**

## 구성 (7장)

| # | 파일 | 슬롯 | 핵심 |
|---|------|------|------|
| 01 | `01-cover.html` | COVER | "자책·몰아붙임·실망이 다 한 순간에" + 5배역 타일 모자이크 배경 |
| 02 | `02-concept.html` | TEXT | 마음은 하루에도 수십 번 바뀐다 — 여러 배역이 번갈아 무대에 |
| 03 | `03-lineup.html` | LINEUP | 다섯 배역 라인업(리더·빌런·난봉꾼·관리자·추방자) + 한 줄 소개 |
| 04 | `04-bridge.html` | TEXT | 나쁜 배역은 없다 — 문제는 자리가 어긋날 때 |
| 05 | `05-imbalance.html` | STAGE | "빌런이 리더 자리에 앉으면" — 무대 중앙 빌런 + 밀려난 배역들 |
| 06 | `06-imbalance-rake.html` | STAGE | "난봉꾼이 나서지 말아야 할 때 나서면" — 무대 중앙 난봉꾼 |
| 07 | `07-result-cta.html` | RESULT+CTA | 제일 중요한 건 내 배역을 알아보는 것 + 「무료/3분」 도장 + "프로필 링크에서 무료 진단 받기" |

## 캐릭터 일러스트 (5배역)

- `illustrations/{leader,villain,rake,manager,exile}.png` — 4:5
- 힉스필드 **Recraft 4.1** 생성 · *어린이 흰 크레파스 낙서 / Mr. Men 그림책 톤* · 흰 선화 on near-black(#0e0e10)
- 표정 방향: 리더(은은한 미소)·빌런(뾰로통)·난봉꾼(폴짝 장난)·관리자(곁눈질 경계)·추방자(웅크림)
- **무대 카드(05·06) 합성:** 캐릭터 배경에 부드러운 회색 글로우가 있어 blend로는
  완전히 안 빠짐 → `illustrations/cutout/*.png`(배경 투명 컷아웃)를 사용한다.
  컷아웃 생성: PIL 휘도 임계(lo=32, hi=110)로 어두운 배경 → 투명, 흰 선만 남김.
  원본 PNG는 표지/라인업/CTA 타일 배경에 그대로 사용(타일 룩 유지).
