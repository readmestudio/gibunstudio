# MINDS · 추방자 (Exile) — IFS 단독 카드뉴스

내면가족체계(IFS)의 **추방자(Exile)** — 상처받은 어린 부분 — 를 다룬 **7장** 인스타 캐러셀.
다크 모노톤(`#0e0e10` + 오렌지 `#e85d3a`). `instagram-cards/shared.css` + `exile.css` 사용.
빌런·리더 단독 시리즈와 동일한 7장 공식. 추방자는 "보호받는 어린 마음"이라 톤을 **더 따뜻·회복적**으로.

## 구성 (7장)

| # | 파일 | 슬롯 | 핵심 카피 |
|---|------|------|------|
| 01 | `01-cover.html` | COVER | "별것 아닌 말에 이상하게 오래 무너진다면" — 공감형 문제 인식 후킹 |
| 02 | `02-intro.html` | 존재 | 누구 마음에나 추방자가 산다 / 다른 마음들이 애쓰는 이유 = 이 어린 나를 지키려고 |
| 03 | `03-examples.html` | 등장 순간 | "혹시, 이런 적 있나요?" 체크리스트 4 (사소한 말에 쿵·거절에 무너짐 등) |
| 04 | `04-behavior.html` | 모습 | 추방자는 이렇게 웅크려 있어요 — 숨어있다/떨고있다/흔들린다/멈춰있다 |
| 05 | `05-thoughts.html` | 말버릇 | "내가 부족해서" / "알면 떠날 거야" / "사랑받을 자격 없어" |
| 06 | `06-understand.html` | 이해 | 추방자는 약점이 아니다 — 가장 보살핌이 필요한 자리, 안아줄 때 회복 시작 |
| 07 | `07-cta.html` | CTA | "내 안의 추방자는 어떤 얼굴?" + 「무료」 테스트 CTA 띠 |

## 에셋 (일러스트)

- `illustrations/exile-cover.png` — 사용자 제공 표지(= `minds-cast/exile.png`). 01·07 공용.
- `illustrations/02~06*.png` — **힉스필드 `nano_banana_pro`** 생성.
  - **표지를 스타일 레퍼런스**(image-to-image)로 걸어 톤 통일: 드라이브러시 오렌지+본화이트 on near-black, 4:5.
  - 02 가슴 속 빛나는 inner child / 03 팔로 막는 움츠림 / 04 작게 웅크린 아이 / 05 무릎에 얼굴 묻은 수치심 / 06 빛나는 아이에게 내미는 따뜻한 손
- 모든 본문 카드는 `illu` 풀블리드 배경 + 스크림(하단 텍스트존 어둡게) + 하단 정렬 텍스트.
