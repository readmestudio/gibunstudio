# 기분 워크북 페이드 소재 — 디자인 규칙

> 상담 워크북 **대기 신청** 유도용 인스타그램 페이드(광고) 소재.
> 다크 + 오렌지 톤 / 실물 워크북 스크린샷 히어로.
> 레퍼런스 구조: `Downloads/Paid_09.png` (상단 라벨 → 히어로 → 하단 굵은 2줄 카피)

---

## 🎨 톤 & 컨셉
- **다크 + 오렌지** / 미니멀 / 실물(워크북 스크린샷) 히어로
- 구조 원리: **상단 라벨 → 밝은 실물 히어로 → 하단 굵은 2줄 카피** (어두운 방에 켜진 스크린 효과 → 시선이 밝은 워크북으로 빨려가고 아래 굵은 카피가 받아침)

## 📐 캔버스
- **1080 × 1350 px** (인스타 4:5 세로형)
- 컨테이너 패딩: `상하 72px / 좌우 80px` (하단만 84px)

## 🧱 레이아웃 (위→아래 3단)

**1. 상단 메타 바** (`space-between`)
- 좌: 브랜드 라벨 `GIBUN · SELF-WORKSHOP` (회색, 대문자, 넓은 자간)
- 우: **대기 상태 뱃지** — 오렌지 테두리 알약형(`border-radius:999px`, `border:1.5px rgba(232,93,58,0.55)`, 패딩 `13px 24px`), 앞에 오렌지 점(12px) + 텍스트 `WAITLIST OPEN` / `COMING SOON`

**2. 히어로 목업** (가운데, 남는 공간 다 차지)
- 워크북 **실물 스크린샷**을 브라우저 카드에 넣음
- 카드: `width 800px`, `border-radius 26px`, 밝은 배경(`#f4f4f5`), 큰 그림자(`0 40px 90px rgba(0,0,0,0.55)`)
- 상단바: 높이 56px 회색, **점 3개 중 첫 번째만 오렌지**(나머지 회색)
- 스크린샷: `height 560px`, `object-fit:cover; object-position:top center`

**3. 하단 카피**
- 메인 카피 2줄, 굵게 흰색 — **핵심 단어 1개만 오렌지**(`<span class="accent">`)
- 서브 카피: 오렌지 점(9px) + `[상품 설명] · [CTA]` 형식, CTA 부분만 흰색 굵게

## ✍️ 카피 규칙 (톤)
- 구조: **1줄 공감(상황) + 2줄 가벼운 제안** — 광고처럼 외치지 않기
- 직장인 3~15년차 페르소나(퇴근·야근·반복 고민), 상담 경험자 페인포인트
- **부드러운 용어**: "고치세요/틀렸어요/반박" ❌ → "살펴봐요/정리하는/다시 보기" ⭕
- 서브 카피 끝은 행동 동사: `지금 사전 대기 신청` / `오픈 알림 먼저 받기` / `사전 신청 오픈`
- 메인 카피 강조색은 **한 단어만** (패턴 / 제자리 / 안 쉬어진다면 등)

## 📋 카피 예시 3종
```
① 패턴      | 같은 고민을 또 반복하고 있다면 / 이번엔 [패턴]부터 살펴봐요
② 상담경험  | 상담은 받아봤는데 / 뭔가 늘 [제자리] 같다면
③ 퇴근후    | 퇴근하고도 마음이 / 잘 [안 쉬어진다면]
```
*([ ] 안이 오렌지 강조 단어)*
