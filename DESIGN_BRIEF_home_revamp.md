## 1. 디자인 방향 (Direction)

### 1-1. 한 줄 요약
**"손그림 모노톤"의 따뜻함은 유지하되, 그 위에 "애플 제품 페이지의 정밀함과 모션"을 얹는다.**

### 1-2. 레퍼런스 톤
- **Apple 제품 페이지** — 거대한 타이포, 충분한 여백, 스크롤에 반응하는 점진적 페이드/리빌, 헤어라인 디바이더
- **Linear / Vercel** — 모노 메타 라벨(`[ EYEBROW ]`), `JetBrains Mono` 보조, 정밀한 미세 보더(1px 헤어라인)
- **이미 프로젝트 내 합의된 톤** — `--v2-*` (인스펙터), `--wb-*` (워크북) 토큰. 홈이 이 톤과 한 가족처럼 보여야 한다.

### 1-3. 살릴 것 / 바꿀 것

| 영역 | 유지 | 변경 |
|---|---|---|
| 색상 | 모노톤 팔레트, 5% 미만 노란 액센트 규칙 | 평면적인 회색 → 톤 단계 분화(`--wb-hair`/`--wb-hair2`/`--wb-hair3` 활용) |
| 타이포 | Pretendard 본문 | 헤딩에 `Inter` + 메타에 `JetBrains Mono` 보조 사용(이미 `globals.css`에 임포트됨) |
| 패턴 SVG | `patternTop` / `patternBottom` 사용 자체 | 풀-블리드 BG → 부분 블리드 + 스크롤 시 패럴랙스 / 페이드 |
| Doodle SVG | 사용 자체 | 정적 배치 → 호버/뷰포트 진입 시 미세 모션 |
| 카드 보더 | 1~2px 보더 | 헤어라인(1px) + 호버 시 `4px_4px_0` 그림자 같은 "툭" 효과는 그대로 유지 가능 |
| 그라데이션 | **금지(`CLAUDE.md` 룰)** — 그대로 금지 유지 | 깊이감은 그라데이션이 아니라 **모션 + 톤 단계 + 헤어라인**으로 표현 |

> ⚠️ **그라데이션 금지 룰은 이번에도 깨지 않는다.** Apple 톤을 그라데이션으로 흉내 내려는 유혹이 있을 수 있으나, 본 프로젝트는 순수 화이트 + 헤어라인 + 모션으로 깊이감을 만든다. 패턴 SVG 위 페이드 마스크(`from-white to-transparent`)는 허용.

## 2. 글로벌 디자인 시스템 업데이트

### 2-1. 활용할 기존 토큰 (globals.css에 이미 존재)
```css
--v2-ink: #0B0D10;        /* 가장 진한 잉크 */
--v2-body: #3A3D44;
--v2-mute: #8B8E96;
--v2-line: #E2E3E7;
--v2-line2: #EDEEF1;
--v2-line3: #F5F6F8;
--v2-paper: #FFFFFF;
--v2-bg: #F7F7F9;

--wb-hair: #E5E5EA;        /* 1px 헤어라인 */
--wb-hair2: #EFEFF2;
--wb-text2: #5B5B62;
--wb-text3: #8A8A92;
```
- 홈 섹션도 이 토큰을 베이스로 통일. 기존 `var(--foreground)` / `var(--border)` 와 병용 가능.

### 2-2. 타이포그래피 스케일 (변경 제안)
| 용도 | 현재 | 제안 |
|---|---|---|
| Eyebrow (메타 라벨) | `text-xs uppercase tracking-widest` | `Inter` / `JetBrains Mono`, 10–11px, `letter-spacing: 0.18em`, `[ MARKER ]` 대괄호 패턴 |
| H1 Hero | `clamp(2rem, 5vw, 3.75rem)` | `clamp(2.5rem, 6.5vw, 5.5rem)`, `letter-spacing: -0.03em`, `line-height: 1.05` |
| H2 섹션 | `md:text-8xl lg:text-6xl` (현재 모바일에서 너무 큼) | `clamp(2rem, 5vw, 4rem)`, `-0.025em` 트래킹 |
| Body | 동일 | 줄간격 `1.65`로 미세 상향 |

> 📌 **카피 자체는 절대 변경 금지.** 줄바꿈(`<br>`)도 가능하면 유지하되, 반응형에서 더 자연스럽게 보이도록 `wordBreak: keep-all` + `clamp` 조합만 손본다.

### 2-3. 모션 토큰
- **이징:** `cubic-bezier(0.22, 1, 0.36, 1)` (애플 표준 부드러움) 또는 `cubic-bezier(0.16, 1, 0.3, 1)` (overshoot 미세)
- **지속시간:** UI 인터랙션 200–300ms / 섹션 진입 600–900ms / 패럴랙스 스크롤 = scroll-driven (시간 무관)
- **`prefers-reduced-motion`** 가드 — `globals.css`에 이미 글로벌 가드 있음. **모든 신규 모션은 이 가드를 통과해야 함.**
- 권장 라이브러리: 이미 설치된 **Framer Motion**. 새 패키지 추가 지양.

## 3. 섹션별 디자인 디렉션

> **각 섹션의 카피·콘텐츠·순서는 그대로.** 아래는 "어떻게 보이게 할 것인가"만 다룬다.

### Section 1 — Hero (`page.tsx` 인라인)

**현재 카피 (그대로):**
- Eyebrow: `심리학 기반 셀프 해킹 플랫폼`
- H1: `인생은 결국 기분 관리,` / `나를 아는 순간 시작됩니다`
- Sub: `심리학 기반 분석으로 나의 기질, 감정, 관계 패턴을 탐색하고 더 기분 좋은 나로 나아가세요.`

**디자인 방향:**
- 풀스크린 BG가 `patternTop.svg` 풀블리드 → 변경: **상단 60% 화이트 + 하단 패턴이 살짝 페이드인**, 또는 **패턴을 1/3 강도로 깔고 위에 헤어라인 그리드 오버레이**.
- H1을 화면 한가운데서 **점진적으로 등장**(스태거: 첫 줄 → 두 번째 줄, 각 80–120ms 딜레이, y: 24→0, opacity: 0→1).
- Eyebrow를 `[ 심리학 기반 셀프 해킹 플랫폼 ]` 모노스페이스 마커 스타일로(텍스트는 그대로, 시각만 마커).
- 스크롤 인디케이터(작은 ↓ + "scroll" 모노 라벨) — 1.5초 후 페이드인, 스크롤 시작하면 페이드아웃.
- **선택 옵션:** Hero 우측 또는 배경 한쪽에 `/public/doodles/` 중 하나(예: `mystic-eye` 또는 `plant-doodle`)를 큰 사이즈로 옅게(`opacity-15`) 배치 + 마우스 무브 시 ±6px 패럴랙스.

### Section 2 — `<ProgramCards />` (3개 프로그램 카드)

**현재 카피 (그대로):**
- 섹션 H2: `나를 해킹하는 프로그램`
- 서브: `혼자서, 또는 상담사와 함께. 진짜 나를 이해해 가는 과정이에요.`
- 카드 3개: `내면 분석 리포트` / `직장인을 위한 마음 챙김 워크북` / `1:1 심리 상담` (제목, 본문, CTA 라벨, `notifyOnly` 상태 모두 그대로)

**디자인 방향:**
- 카드 그리드 → 동일하게 3열 유지하되 카드 자체를 **헤어라인(1px `--wb-hair`) + 화이트 페이퍼** 톤으로 정리.
- 일러스트(96×96 doodle)는 **호버 시 ±3deg 회전 + 8px y 부유**(Framer Motion `whileHover`).
- CTA `검사 선택하기 →` / `구매하기 →` 의 화살표를 호버 시 **+4px 우측 슬라이드**.
- `notifyOnly` 카드(opacity-70)는 유지하되, 호버 시 미세하게 명도 회복(0.7→0.85).
- 카드 진입 모션: 뷰포트 진입 시 좌→우 스태거(120ms 간격), y: 24→0.
- 섹션 상단에 **얇은 헤어라인 + 모노 인덱스** 추가(예: `[ 01 / PROGRAMS ]`) — 카피 추가가 아니라 번호/구분 마커. 이게 카피 변경에 해당한다고 판단되면 생략 가능.
  > ⚠️ "카피 추가"로 해석될 여지가 있으니 **이 마커는 옵셔널.** 디자이너 판단으로 추가/제거 결정.

### Section 3 — `<ComparisonSection />` (Old Way vs New Way)

**현재 카피 (그대로):**
- H2: `AI 시대에 가장 중요한 건` / `나를 알아가는 것입니다`
- 좌측 헤더: `혼자서 고민할 때` + 5개 항목
- 우측 헤더: `기분 스튜디오 셀프 해킹` + 5개 항목

**디자인 방향:**
- BG `patternBottom.svg` + 화이트 80% 오버레이 → **유지하되 패턴을 더 옅게(50–60%)**, 그리고 스크롤 시 패턴이 위→아래로 천천히 패럴랙스.
- 좌/우 카드를 **이중 비교 테이블처럼**: 좌측은 회색 톤(`--v2-line3` 배경), 우측은 화이트 + 검정 보더 2px. 이미 비슷하지만 **두 카드 사이에 세로 헤어라인(1px) + 중앙에 작은 "vs" 뱃지** 추가.
- 항목 진입: 좌/우 카드가 동시에 뷰포트 진입할 때, 좌측은 좌→우 슬라이드, 우측은 우→좌 슬라이드(서로 마주보며 합쳐지는 느낌, 400ms).
- `+`(좌측) / `✓`(우측) 아이콘은 그대로. 우측 ✓ 만 진입 시 **draw-on SVG 애니메이션**(stroke-dasharray 0→100%, 600ms 스태거).
- 모바일에서는 좌측 카드를 위, 우측을 아래로(현재 동일). 모바일에서도 페이드인 스태거만.

### Section 4 — `<FeatureTwo />` (3-Step + 폰 목업 슬라이드)

**현재 카피 (그대로):**
- H2: `Hack yourself.` / `Freedom begins.` (영문 그대로 유지)
- 서브: `나를 이해하는 순간, 선택할 수 있게 돼요.`
- 좌측 3개 번호 항목 (1, 2, 3) — 텍스트 전부 그대로
- 우측 폰 목업 + 4장 리포트 카드 (`Type` / `스트레스 반응` / `관계 인사이트` / `매칭 결과`)

**디자인 방향:**
- H2 `Hack yourself. / Freedom begins.` — **`Inter` 폰트 + 매우 큰 타입(`clamp(3rem, 8vw, 6rem)`) + `-0.04em` 트래킹**. 이 섹션이 홈에서 가장 "테크 매니페스토"한 순간.
- H2 진입: 두 줄을 각각 mask-clip 슬라이드(아래에서 위로 마스크 해제, 700ms, 두 번째 줄 200ms 딜레이).
- 좌측 번호(1/2/3) 원형 배지 → **검정 원형 + 안에 `Inter Bold`**. 배지 사이에 세로 헤어라인(`--wb-hair`)을 추가해 "타임라인" 느낌. 이미 비슷한 구조(`pb-10`)이지만 시각적으로 **연결선 강조**.
- 우측 폰 목업:
  - 노치/베젤은 모노톤 그대로 유지(검정 4px 보더).
  - 슬라이딩 카드 자동 스크롤은 유지하되, **속도 20s → 30s** (애플 톤은 더 느리고 천천히).
  - 폰을 **마우스 무브 시 미세 3D 틸트**(±4deg, perspective 1000px). reduced-motion이면 비활성.
  - 폰 주변에 작은 doodle 1–2개를 **떠다니는 액센트로** 배치(`star-sparkle.svg`, `heart-doodle.svg` 등) — 매우 옅게(opacity 0.2), 천천히 회전(20s linear infinite).

### Section 5 — `<SelfHackingElements />` (4단계 커리큘럼)

**현재 카피 (그대로):**
- H2: `패턴을 깨고 나로 살아가는 여정`
- 서브: `나를 이해하고, 관계를 해독하고, 삶에 적용하는 전체 여정이에요.`
- 4개 단계: `나에 대한 이해` → `관계 속의 나` → `실전 적용` → `완전한 얼라인` (모든 description, tag 그대로)

**디자인 방향:**
- 현재 4개 카드 + chevron 화살표 구조 → **타임라인 스테퍼**로 진화:
  - 데스크탑: 가로 라인이 4개 노드를 잇는 형태. 노드 = 원형 헤어라인 + 안에 SVG 아이콘. 라인은 1px `--wb-hair`.
  - 뷰포트 진입 시 **라인이 좌→우로 그려지는 애니메이션**(scaleX 0→1, 1200ms, transform-origin: left).
  - 노드는 라인 그리기 진행에 따라 순차적으로 **fill-in**(투명 → 검정 보더 → 아이콘 페이드인). 각 노드 진입 시 미세한 ring pulse(`globals.css`의 `pulseRing` 키프레임 재활용 가능).
- 4번째 노드(`완전한 얼라인`)에 도달했을 때 **체크 아이콘이 마지막에 draw-on**되며 강조.
- 모바일에서는 세로 타임라인. 라인이 위→아래로 그려지면서 노드 등장.
- 카드 박스 자체는 단순화: 보더 → 헤어라인, 그림자 제거, 호버 시 노드만 미세하게 scale(1→1.05).

### Section 6 — `<Testimonials />` (3열 매소너리 리뷰)

**현재 카피 (그대로):**
- H2: `"나를 알겠다"가 된 순간들`
- 서브: `막연했던 내가, 선명해진 사람들의 이야기예요.`
- 6개 리뷰 (이름 이니셜, 전체 텍스트 그대로)

**디자인 방향:**
- 3열 정적 그리드 → **수직 스크롤 마퀴 3열**. 좌측 열은 위→아래로, 가운데 열은 아래→위로, 우측 열은 위→아래로 천천히 흐름. 속도 60–90s. 호버 시 일시정지.
- 카드 자체:
  - 보더 + 그림자 → 헤어라인(1px) + 호버 시 미세 lift(translateY -2px, shadow soft).
  - 이니셜 원 → **검정 원 그대로 유지**. 카드별로 미세하게 다른 회색 톤(체스판처럼 살짝 분화) — `--v2-line3`/`--v2-line2`/`--v2-paper` 로테이션.
  - 리뷰 본문 폰트는 Pretendard 그대로. 단, 따옴표(`"…"`)를 큰 별도 큰 모티프(`Inter Italic`, `8em`, opacity 0.06)로 카드 좌상단에 배경 워터마크.
- 섹션 상하단에 **페이드 마스크**(8% from top, 8% from bottom) — 마퀴가 자연스럽게 끊기게.
- 모바일: 마퀴 단일 열로 축소(또는 가로 스와이프). 자동 스크롤 속도 더 느리게.

**현재 카피 (그대로):**

- 서브: `1급 심리 상담사와 Zoom으로 진행하는 전문 상담`
- 4개 플랜 카드 (`COUNSELING_TYPES` from `src/lib/counseling/types.ts`) — `BEST` 뱃지 포함 모두 그대로
- CTA: `알림 신청` (모든 카드)

**디자인 방향:**
- 4-up 그리드 유지. 카드 → **애플 가격 비교 테이블 스타일**:
  - 보더 헤어라인(1px), 라운드 16–20px.
  - `BEST` 카드만 **검정 2px 보더 + 미세한 ring shadow**(그림자가 검정 8% 정도, 24px blur). 그라데이션 X.
  - 플랜 제목(eyebrow) → 모노 라벨 스타일(`Inter` / `JetBrains Mono`, 트래킹 0.18em).
  - 가격 숫자(`text-4xl`) → **`Inter Bold`로 교체, `clamp(2.5rem, 4vw, 3.5rem)`, `-0.03em` 트래킹**. 단위(`원 / 50분`)는 모노 작게.
  - 체크/크로스 아이콘은 그대로. 단, 체크는 `--v2-ink`, 크로스는 `--v2-mute2`로 톤 분리.
- 카드 진입: 좌→우 스태거(80ms 간격, y: 16→0).
- `BEST` 뱃지: **subtle pulse**(2s 주기, scale 1↔1.04). reduced-motion 가드 적용.
- CTA `알림 신청` 버튼: 호버 시 **검정 보더 → 검정 채움 인버트**(220ms, 모노톤이라 그라데이션 없이도 깔끔). `BEST` 카드 CTA는 이미 검정 채움이므로 호버 시 미세하게 70% 명도 변경.
