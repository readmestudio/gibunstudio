## 1. 디자인 시스템 — Monotone 톤

### 1-1. 컬러 토큰 (CSS 변수)

```css
--background: #ffffff;     /* 페이지 배경 */
--foreground: #191919;     /* 텍스트, 보더, 검정 fill */
--surface: #fafafa;        /* 약한 회색 배경 (카드, 인용 박스) */
--accent: #FFE812;         /* 노란색 — 5% 미만, New 뱃지 등 특수 케이스만 */
--border: #e5e5e5;
```

### 1-2. 디자인 원칙

- **그라데이션 금지** — 순수 화이트 베이스
- **검정색은 텍스트/테두리만** — 강조 시 검정 fill 박스 (`bg-[var(--foreground)] text-white`) 사용
- **불투명도(opacity) 활용** — `text-[var(--foreground)]/70`, `/65`, `/55`, `/45`, `/40`, `/25` 단계로 위계 표현
- **테두리 강조** — `border-2 border-[var(--foreground)]` (강) / `border-2 border-[var(--foreground)]/12` (약)
- **둥근 모서리** — `rounded-2xl` (큰 카드), `rounded-xl` (중간), `rounded-full` (CTA, 칩, 아이콘 원)
- **그림자** — `shadow-[3px_3px_0_rgba(0,0,0,0.06)]` (작은 옵셋 검정 shadow, 도장 같은 느낌)
- **폰트** — Pretendard (한국어 본문), Georgia/Times serif (인용구·확언), JetBrains Mono (라벨 일부)

### 1-3. 타이포 위계 (대표 클래스)

| 위계 | 사이즈 |
|------|--------|
| H1 (Hero) | `text-4xl sm:text-5xl md:text-6xl font-bold` |
| H2 (섹션 헤드라인) | `text-2xl sm:text-3xl md:text-4xl font-bold` |
| H3 (서브 헤드라인) | `text-xl sm:text-2xl font-bold` |
| 본문 | `text-base sm:text-lg leading-relaxed` |
| 캡션 | `text-sm text-[var(--foreground)]/65` |
| Eyebrow (섹션 라벨) | `text-xs font-semibold tracking-widest uppercase text-[var(--foreground)]/40` |
| Mono 메타 | `tabular-nums tracking-widest uppercase` |

### 1-4. 공통 인터랙션

- **CTA 버튼** — `rounded-full bg-[var(--foreground)] text-white px-7 py-4 font-bold` + `hover:scale-[1.02] active:scale-[0.98]`
- **카드 hover** — `hover:border-[var(--foreground)]/30` (보더 진해짐)
- **break-keep** — 한국어 본문 거의 모든 곳에 적용 (단어 단위 줄바꿈)

#### 1:1 심리 상담의 한계 카드 그리드 (4개)

- **헤드라인 (3줄)**:
  > 1:1 심리 상담의
  > 이런 문제를
  > 해결하고 싶었습니다
- **카드 그리드** — `grid sm:grid-cols-2 lg:grid-cols-4` 4개 카드
- **카드 내부** (각 카드 동일 구조):
  - 원형 아이콘 (보더만, fill 없음, h-12 w-12)
  - 카드 제목 (`text-base sm:text-lg font-bold`)
  - 카드 설명 (`text-sm text-[var(--foreground)]/65`)

| # | 아이콘 (의미) | 제목 | 설명 |
|---|--------------|------|------|
| 1 | coin (동전, $) | 가격이 부담돼요 | 회당 8~15만원, 매주 가는 건 현실적으로 부담돼요. |
| 2 | loop (순환 화살표) | 같은 얘기만 반복하게 돼요 | 상담이 여러 회차 진행되면 어느 순간 같은 문제, 같은 이야기를 반복하고 있는 것 같아요. |
| 3 | mismatch (사람 + X) | 상담사가 잘 맞지 않아요 | 상담사가 나와 잘 맞냐 안 맞냐에 따라 심리 상담의 만족도가 달라지더라구요. |
| 4 | empty (빈 봉투) | 상담이 끝나고 남는 게 없어요 | 이야기할 땐 신나게 떠들었는데, 끝나고 남는 게 없어요. |

- **카드 그리드 결론 카피** (밑줄 강조):
  > 문제는 상담사가 아니라, **상담의 구조와 도구**에 있었습니다.

#### 검정 박스 — 워크북의 출발점

- 큰 검정 fill 박스 (`rounded-2xl`, 흰 텍스트, 두꺼운 본문)
  > 심리 상담사 없이도 인지 행동 치료에 기반하여
  > 상담을 대신할 수 있는 워크북은 없을까?
- 박스 하단 흐린 캡션:
  > 마음 챙김 워크북은 이 질문으로부터 시작되었습니다.

#### 모션

- 슬라이드쇼: `framer-motion` `animate={{ x: ["0%", "-50%"] }}` 70초 무한 linear loop
- 다른 요소: 모션 없음 (정적)

#### 우측 카드 — 마음 챙김 워크북 (After, 강조)

- 제목: **마음 챙김 워크북** (흰색)
- 마커: ✓ 아이콘 (흰 fill 원, 검정 체크)

#### 비교 항목 6줄 (좌·우 동일 라벨)

| 라벨 | 일반 상담 | 마음 챙김 워크북 |
|------|----------|-----------------|
| 비용 | 회당 8~15만원 | 한 번 구매 3만원대 |
| 시간 | 매주 예약·이동·50분 | 내가 원할 때, 원하는 속도로 |
| 반복성 | 같은 이야기 맴돌기 쉬움 | 구조화된 단계로 진전 |
| 상담사 | 맞고 안 맞음이 큼 | 변수 없음 |
| 시작과 끝 | 연장·종결 얘기 꺼내기 부담 | 내가 원할 때 시작하고 멈춘다 |
| 결과물 | 기억·메모에 의존 | 분석 리포트 3종이 남음 |

#### 가운데 화살표

- 검정 fill 원형 (h-12 w-12), 흰색 → 화살표
- **모션** — `animate={{ x: [0, 4, 0] }}` 1.6초 반복 (좌우로 살짝 뛰어다님)
- 모바일에선 90도 회전 (`rotate-90`) → 위→아래 화살표

#### 카드 진입 모션

- 좌측: `initial={{ opacity: 0, y: 16 }}` → `animate` (delay 0)
- 우측: 동일 모션, **delay 0.2s** (좌→우 순차 등장)
- 둘 다 `whileInView`, `viewport.amount: 0.3`, `duration: 0.55`

#### FAQ 8문항

##### Q1. 심리 상담을 받아본 적 없는데 혼자 할 수 있나요?

네, 상담 경험이 없어도 따라갈 수 있도록 설계되어 있습니다. 전문가가 할 법한 질문이 순서대로 준비되어 있고, 예시와 작성 가이드가 단계마다 붙어 있어요. 혼자서도 막히지 않고 진행할 수 있도록 구조화되어 있습니다.

##### Q2. 우울증/불안장애 진단을 받은 사람도 사용할 수 있나요?

마음 챙김 워크북은 치료 목적이 아니며, 스스로의 마음을 이해하고 대처하는 도구입니다. 이미 진단을 받으셨다면 전문의·상담사 상담과 병행해서 사용해 주세요. 워크북에 정리된 패턴과 기록은 전문가와 이야기할 때 유용한 자료가 될 수 있습니다.

##### Q3. 워크북 완성까지 얼마나 걸리나요?

한 번에 몰아서 하면 약 65~100분 정도 걸립니다. 중간에 저장하고 며칠에 걸쳐 진행해도 괜찮아요. 콘텐츠 조회 기간은 결제 후 90일이며, 그 안에 내가 원하는 속도로 진행할 수 있습니다.

##### Q4. 효과가 없으면 환불되나요?

1단계 진단을 시작하기 전까지는 전액 환불이 가능합니다. 진단 이후에는 디지털 콘텐츠 특성상 환불이 제한될 수 있어요. 먼저 카카오톡 gibunstudio로 문의해 주시면 상황에 맞춰 안내드립니다.

##### Q5. 성취 중독과 불안 둘 다 해당되는데 어떤 걸 사야 하나요?

지금 가장 자주, 가장 강하게 일상에 영향을 주는 주제를 먼저 선택하시길 권합니다. 두 주제는 서로 겹치는 부분이 있어서 한 권을 끝내면 다른 주제도 이전보다 잘 들여다볼 수 있게 돼요. 두 권 모두 진행해도 좋습니다.

##### Q6. 다 쓴 후에 재구매 없이 다시 쓸 수 있나요?

결제 후 90일 동안은 언제든 다시 열어 내용을 확인하고 재작성할 수 있습니다. 같은 주제를 더 오래 쓰고 싶다면, 출력해 두거나 별도 문서에 옮겨 두는 것을 권해요.

##### Q7. 전문가 상담은 꼭 받아야 하나요?

꼭 받으실 필요는 없어요. 워크북만으로도 진단 → 실습 → 대처까지 완결됩니다. 작성한 내용이 스스로 정리되지 않거나 객관적인 시선이 필요할 때, 선택적으로 1회성 해석 상담을 이용하실 수 있습니다.

##### Q8. 개인정보(내가 쓴 내용)는 어디에도 공유되지 않나요?

작성하신 워크북 내용은 본인만 열람할 수 있도록 저장됩니다. 홍보·마케팅에 활용되지 않으며, 제3자에게 제공되지 않아요. 전문가 해석 상담을 신청하신 경우에 한해, 담당 상담사와만 안전하게 공유됩니다.

## 6. 리뱀프 시 가이드

### 6-1. 보존 (절대 바뀌면 안 됨)

- 모든 한국어 카피 텍스트
- 13개 섹션 순서
- 각 모션의 **의도와 의미** (구체적 표현은 자유)
- CTA 링크: `#waitlist`
- 워크북 그리드 섹션의 `id="workbooks"` 앵커 (스티키 버튼 타겟)
- 가격 ₩49,000 (소프트 런칭 특가)
- 보안 3가지 항목 (Row Level Security, HTTPS, Authenticated Session)

### 6-2. 자유로운 영역 (디자이너 재량)

- 각 섹션의 레이아웃 비율 (좌/우 split, grid columns 등)
- 컬러 톤 — Monotone 베이스만 유지하면 검정/흰/회색 비율은 자유
- 타이포 사이즈 위계 (현재는 `text-2xl ~ 6xl` 범위)
- 모션 표현 (예: 마퀴 → 클릭으로 다음, fade-up → spring 등)
- 카드 모서리, 그림자 깊이, 보더 굵기
- 아이콘 스타일 (현재는 stroke 기반 SVG)

### 6-3. Monotone 디자인 가이드 재확인

- 그라데이션 금지
- accent(노란색) 5% 미만
- 검정색은 텍스트/테두리/강조 fill만
- `bg-[var(--foreground)]` (검정 fill) 카드는 한 페이지에서 **3~4개 이내**로 사용 (현재 4개: SoftLaunchBanner / Hero 검정 박스 / 가격 우측 카드 / Privacy 카드)
